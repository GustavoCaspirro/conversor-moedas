Sobre

Conversor de moedas criado com vue.js utilizando API pública ("https://economia.awesomeapi.com.br/all/[moedaB]-BRL"), onde o parametro "moedaB" é o tipo de moeda, exemplo: USD.

A aplicação possibila comparações entre moedas.
