# Conversor de moedas (VUE.js)
---

## Sobre
Conversor de moeda simples consumindo uma api.

## Passos para roder o projeto

### Clonar o projeto
```
$ git clone projetoClonado
```

### Rodar o projeto
```
$ cd projetoClonado
$ npm install
$ npm run serve
```

### Compiles and hot-reloads for development
```
npm run serve
```
